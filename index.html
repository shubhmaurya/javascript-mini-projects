Dynamic Color Changer

In this JS script listens for clicks on buttons representing different colors. When clicked, the background color of the page changes dynamically.

BMI Calculator

Here JS script handles form submission for inputting height and weight. Upon submission, it calculates the Body Mass Index (BMI) and displays the result.

Digital Clock

Using the `setInterval` function, this script continuously updates the displayed time on the webpage to reflect the current time.

Number Guessing Game

Here JS script lets users guess a random number between 1 and 100. It gives feedback on whether the guess is high or low, ending the game after 10 attempts or upon guessing the correct number.

Know Your KeyCode

In this JS script listens for keydown event listener. Here when clicked, it returns the KeyCode or character code of pressed key.

Unlimited Colors with Time

In this JS script generates random color and set it to the background and have the time control of color changing speed. Also adjust the text color with respect to brightness.
